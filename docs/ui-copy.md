# UI microcopy & labels

## Buttons
- Primary action: **Book Appointment**
- Secondary: **Save**, **Cancel**, **Preview**
- Small action: **Reschedule**, **Add staff**

## Form field labels & placeholders
- Name → placeholder: `Enter full name`
- Phone → placeholder: `+91 91234 56789`
- Email → placeholder: `name@example.com`
- Notes → placeholder: `Any requests? (optional)`

## Success messages
- Booking success: `Booking confirmed. A confirmation email was sent to {{customer_email}}.`
- Save success: `Changes saved.`

## Error messages (short, actionable)
- `Unable to fetch availability. Try again in a few seconds.`
- `Selected time is no longer available. Please pick another slot.`
- `Please enter a valid email address.`

## Tooltips & helper text
- For buffer time: `Time added after each appointment for cleanup.`
- For skills: `Skills determine which staff can perform the service.`

## Accessibility note
- Use clear labels, avoid ambiguous verbs like "Submit" when possible.
