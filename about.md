# About This Project

This demo was built to showcase SaaS technical writing skills aligned with real-world documentation standards.

**Highlights:**
- Persona-driven content and user-centered instructions.
- Voice & tone consistency across guides.
- Demonstration of Git-based documentation workflows.
- Example collaboration and review process.

**Technologies used:** HTML, Markdown, GitHub Pages.

**Purpose:** To simulate real-world product documentation for the wellness SaaS domain.
